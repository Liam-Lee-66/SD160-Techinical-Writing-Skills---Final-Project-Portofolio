What is HTML & CSS?

Author: Liam Lee

Published: 12/08/2021

What is HTML?

To make a website there are several processes. One of the processes is the programmer communicating with the browser how their website should look like. However, the browser does not speak the human language, so the programmer can’t communicate with the server with the human language. "Then how will we make our website?" You may ask. For that to be possible we use something called Hypertext Markup Language (HTML). HTML is a programming language that we use to tell the browser what we want on our website. As an example, if you wish to have a Title, Paragraph, or even an image on your website, you will be using HTML to tell the browser what you want.

What is CSS?

Have you ever seen a website without CSS in 2021? I bet you haven’t. Here’s the reason why:

To not make an old and bad-looking website, we use something called Cascading Style Sheets (CSS). While HTML shows how the content of the website should look, CSS edits how the HTML should be shown in a browser. In very simple words, CSS changes how the HTML looks.

Let's get deeper into it!

While we are programming, we must let the browser know what we want to add, edit, or even change. To communicate with the browser, we use something called HTML tags. HTML tags tell the browser which specific element we wish to edit in our website. HTML tags contain three main parts: an opening, the content, and the closing tag.

Here’s how they look like (red = opening, blue = content, green = closing):

With these tags, you can add whatever you want in HTML. Here’s a small tip from me; As you may have already realized, there is a very easy way to indicate if a tag is an opening tag or a closing tag. Simply look for a “/” in front of the tag name, this means it’s a closing tag.
